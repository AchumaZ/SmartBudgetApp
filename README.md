# SmartBudget App

## Overview
SmartBudget is a personal budgeting Android prototype developed in Kotlin.

## Features
- Login screen
- Dashboard navigation
- Expense tracking
- Categories
- Budget goals
- Room database implementation

## Technologies
- Kotlin
- Android Studio
- RoomDB
- GitHub

## Logging
Application includes Logcat logging for major screens.
